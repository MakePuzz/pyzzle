pyzzle

puzzle-japan presents

A Python library to automatically generate intelligent and beautifull puzzles

Introduction

pyzzle は puzzle-japan が提供するスケルトンパズル生成ライブラリです。

これまでにない程の知的で美しいパズルを自動的に生成します。

LICENCE

pyzzle は MITライセンスに準拠します

Install

  • pip install

    pip install pyzzle

  • conda install

    conda install pyzzle

  • GitHub install

    git clone https://github.com/puzzle-japan/pyzzle.git
     cd pyzzle
     python setup.py install
