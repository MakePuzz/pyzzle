pyzzle

Introduction

pyzzleは，puzzle-japanが提供するスケルトンパズル生成ライブラリです。

Install

  • pip install

    
                    pip install pyzzle

  • conda install

    
                    conda install pyzzle

  • GitHub install

    
                    git clone https://github.com/puzzle-japan/pyzzle.git
                    cd pyzzle
                    python setup.py install
